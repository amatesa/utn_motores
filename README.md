# Prototipo de Videojuego - Motores I

### Descripción

Prototipo de videojuego desarrollado en Unity para la materia Motores de Videojuegos I.  
Incluye mecánicas básicas de movimiento, combate y enemigos, aplicando conceptos de prototipado iterativo y core loop.

---

### Objetivo del Proyecto

Desarrollar un prototipo jugable que permita validar mecánicas principales del gameplay antes de una versión final.

---

### Controles

- Movimiento: WASD o Flechas
- Rotación: Mouse
- Acciones: (a completar)

---

### Mecánicas Implementadas

- Sistema de movimiento del jugador
- Sistema de vidas (3 vidas)
- Enemigos que persiguen al jugador
- Sistema de daño
- Condición de victoria y derrota
- Reinicio de partida
- Uso de físicas (Rigidbody)
- Uso de Raycast
